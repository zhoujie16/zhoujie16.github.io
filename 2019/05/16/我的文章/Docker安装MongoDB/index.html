在上面的命令中，几个命令参数的详细解释如下：
-p 指定容器的端口映射，mongodb 默认端口为 27017
-v 为设置容器的挂载目录，这里是将即本机中的目录挂载到容器中的/data/db中，作为 mongodb 的存储目录
--name 为设置该容器的名称
-d 设置容器以守护进程方式运行
